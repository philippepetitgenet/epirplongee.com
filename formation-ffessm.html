Découverte

Baptême

Chaque baptisé est accompagné par un moniteur qui se charge de présenter l’organsiation du baptême sous forme de petit briefing puis d’accompagner sous l’eau la personne.

- A partir de : 8 ans

- Nombre de séances : 1

- Tarif : 50 €

Plongeur de Bronze

Le Scuba Diver vous a plu mais vous ne savez pas encore si vous voulez vous lancer dans l’Open Water, le brevet Junior Scuba Diver vous donnera le tout premier niveau de plongeur sous-marin pour profiter rapidement de vos prochaines plongées loisirs

- Dilpôme : Plongeur de bronze

- A partir de : 8 ans

- Nombre de séances : 3
